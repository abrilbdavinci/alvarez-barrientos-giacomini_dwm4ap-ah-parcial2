# Kälm - DWM4AP P2 AH

Este proyecto es una aplicación web que permite explorar productos y marcas pensadas para la rutina diaria. Incluye funcionalidades para ver productos, filtrar por marca o tipo, y crear nuevas marcas directamente desde la interfaz. Está desarrollado con **React** en el frontend y consume una API REST para obtener los datos.

---

## Contenido del proyecto

- **src/**: Carpeta principal del código fuente.
  - **views/Home.jsx**: Vista principal donde se muestran los productos y el sidebar con marcas.
  - **components/**: Componentes reutilizables como `Card`, `Form` y `ListSimple`.
  - **utils/AuthContext.jsx**: Contexto de autenticación y manejo de token.
- **index.css**: Estilos globales del proyecto.
- **App.jsx**: Configuración de rutas y componentes principales.

---

## Requisitos

- Node.js >= 18
- npm o yarn
- Backend API disponible en `http://localhost:3000/api`
